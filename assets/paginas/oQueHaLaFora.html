Visão Geral

Não há "top" para a World-Wide Web. Você pode olhar para ele de muitos pontos de vista. Se você não tem outro viés, aqui estão algumas maneiras de procurar informações.

por Assunto

  • Uma classificação por assunto de interesse. Incompleto, mas mais fácil de usar

por Tipo

  • Procurar por tipo de serviço (protocolo de acesso, etc) pode permitir encontrar coisas se você souber o que está procurando.

Se você tiver que usar um nó "top", recomendamos este nó ou a lista de assunto. Veja também: Sobre o projeto W3.
