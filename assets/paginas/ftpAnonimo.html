Obtendo os arquivos WWW

Leia isso se você ainda não tem os arquivos WWW que você precisa e você não pode montar um diretório no qual eles residem.

A fonte e o binário dos arquivos do projeto WWW estão atualmente disponíveis (ver copyright ) por FTP anônimo do nó info.cern.ch. Este nó é atualmente 128.141.201.74, mas ESTA MUDANÇA DE MAIO. Os arquivos são arquivos de fita compactada (.tar. Z). Para obter um arquivo usando FTP anônimo, use o nome de usuário "anônimo" e seu endereço de e-mail como senha:-

> ftp info.cern.ch Digite o nome do usuário: anônimo Senha: me@myhost.edu ftp> cd /pub/www/src ftp> ftp binário> obter ftp WWWLineMode_vvv.tar.Z ftp> sair >

O nome do arquivo dependerá do produto que você deseja. Aqui vvv é o número de versão do software.

Uma vez que você tenha o arquivo de piche comprimido, você deve descompactá-lo e desembrulhá-lo.

  • descomprimir WWWLineMode_vvv.tar.Z

Isso fará um arquivo um pouco maior com o nome sem o ". Z". Para desembrulhá-lo,

  • piche xvf WWWLineMode_vvv.tar

Agora você deve instalar o navegador de modo de linha, instalar o servidor, instalar Viola, etc.
