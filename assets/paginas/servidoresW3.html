Webmaster

Esta é uma lista de alguns servidores WWW. Ele não inclui todos os servidores, e note que uma máquina de servidor pode servir muitos bancos de dados. Veja também: plano de fundo em WWW , e dados disponíveis por outros protocolos, dados por assunto, como fazer um novo servidor, servidores de teste. Se os servidores são marcadores "experimentais", você não deve esperar nada. O topo da lista está em ordem cronológica inversa de adição.

NCSA

  • Centro Nacional de Aplicações de Supercomutura, Urbena Champain, IL, EUA, Experimental.

IN2P3

  • Lyon, França.

KVI

  • Kernfysisch Versneller Instituut (instituto acelerador de física nuclear), Groningen, Holanda. Servidor VMS.

CWI

  • Centro de Matemática e Ciência da Computação, Amsterdã. Servidor FTP para hipertexto, incluindo material Gnu TeXInfo como hipertexto.

Cornell

  • Informações legais: Status de Propriedade Intelectual dos EUA on-line.

ZEUS

  • Experimento ZEUS em DESY, Hamburgo, Alemanha. [. Pelo menos 2 servidores]

KEK

  • KEK, Tsukuba, Japão. Só experimental. [FTP hipertexto, servidor http mais tarde)

Servidor unix DESY

  • Experimental ainda.

Biblioteca Técnica da Dinamarca

  • O serviço de informações DTB inclui o sistema de biblioteca.

Revista VOICE

  • A primeira revista global de hipertexto online? Ed. Tom Boutell

SLAC

  • Acelerador Linear de Stanford, Califórnia. HEP pré-prime banco de dados e MUITO mais... também um servidor linux.

Universidade Técnica de Graz

  • Serviço de informação. Gateway para dados Hyper-G.

CCIT Arizona

  • Informações da Universidade do Arizona.

Fermilab

  • Dicumentação de grupos on-line e off-line. Também na FNAL, servidores muito experimentais em Teoria, D0, gerenciamento HEPnet, consultoria de usuário ACCESS.

Repositório de software ASIS

  • Um servidor para domínio público e software CERN para distribuição apenas aos membros do CERN. A documentação é pública.

Notícias do CERN

  • Vários grupos, alguns mais ativos do que outros - veja a lista completa

Documentos de computação CEM

Universidade Hebraica de Jerusálem

  • HUT Árvore de informações.
